测试数据集
位置
测试数据集存放在 data/test_dataset/ 目录下。

实验权重文件
位置
实验权重文件存放在 weights/ 目录下。

描述
实验权重文件包含若干模型权重文件。这些文件是通过不同实验得到的模型权重，用于模型的加载和推理。
